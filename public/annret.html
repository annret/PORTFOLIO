Details

Photography

html, css, webdesign

"Photography" is a project I made at the university of applied sience. The task was to program a website only with HTML and CSS which includes one main page and at least three subpages. Since one of my favorite hobbies is to photograph, I chose this topic and named the website "Photography".

03

Movie Trip Map

html, css, js, php, webdesign

"Movie Trip Map" is a web application which enables you to find out where a movie was filmed and provides detailed information about the movie and the selected country. The aim of this website is to address as many people as possible who enjoy traveling and movie nights and to give them more knowledge about the location and the film.
