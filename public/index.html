about me

Hi! My name is Anna, I'm 23 years old and I live in Salzburg, Austria. Currently I am studying Communication and Knowledge Media (MA) at the University of Applied Siences in Upper Austria. Before I did my bachelor's (BSc) in Multimedia Technology with focus on web development. Below you will find a few projects and more information about me. Have fun and let's connect!

Details

Details was one of my first web-projects I have made. We made it in school with the frontend-web-framework Bootstrap. Details is a website which showes the beauty of nature with little quotes beside a few pictures which I took by myself.

html - css - js

02

Photography

Photography is a project I made at the University of Applied Siences. The task was to program a website only with HTML and CSS which includes one main page and at least three subpages. Since one of my favorite hobbies is to photograph, I chose this topic and named the website "Photography".

html - css

03

Movie Trip Map

Movie Trip Map is a web application which enables you to find out where a movie was filmed and provides detailed information about the movie and the selected country. The aim of this website is to address as many people as possible who enjoy traveling and movie nights and to give them more knowledge about the location and the film.

html - css - js - php

04

Fast and Good

FAGO is the name of our University's canteen. As the old website was made in 2010, my team (two colleagues) and I decided to make a new one. The required programming-language, we had to use, was ruby on rails. This project was made in one intensive week at University, which is called "studioweek", where we get to know the work-life.

html - sass - js - ruby on rails

05

StudiesHub

Finding a Co-working space and reserve a specific desk to work with your colleagues and friends has never been easier. With StudiesHub, which is a room/table-reservation tool, it is simple to book a table in one of the co-working rooms at the FH Salzburg. My colleague's and my requirement was, to make it easy to understand and appealing for the user.

html - sass - js - php
